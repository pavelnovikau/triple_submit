Operation Mode

How the extension processes Enter key presses

Normal Mode

In Normal mode, the extension counts consecutive Enter key presses and only allows form submission when the required number is reached. The form submits normally using the browser's default behavior.

  • Best for most websites
  • Compatible with standard forms
  • Works with regular input fields and textareas

Alternative Mode

In Alternative mode, the extension uses a different technique to detect and prevent form submissions. The extension blocks all standard Enter key submissions and only allows submission after the required number of presses.

  • Better compatibility with complex web applications
  • Suitable for sites where normal mode doesn't work
  • More aggressive interception of form submissions
  • Recommended for Shift+Enter key combinations
